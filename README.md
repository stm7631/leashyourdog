# Leash Your Dog

A tiny shop from San Francisco. 100% of proceeds go to Bay Area dog rescues.

**Live site:** [leashyour.dog](https://leashyour.dog)

## Structure

- `index.html` — Desktop experience. Cursor carries a gravity leash; hover over the dog to catch it.
- `mobile.html` — Mobile experience. Draw a lasso around the dog to catch it. Auto-redirected from `index.html` on touch devices.
- `tokens.js` — Design tokens, font stacks, product data, and copy. Edit here to retheme or update products.

## Hosting

Deployed via GitHub Pages from the `main` branch root. Custom domain configured via `CNAME`.
